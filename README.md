# 🍔 Foodie - Responsive Food Ordering Website

A modern and responsive Food Ordering Website built using **HTML5, CSS3, and JavaScript**. The project provides an attractive user interface for browsing food items, adding products to the cart, reading customer reviews, and exploring food delivery services.

---

## 🚀 Features

- Responsive and modern UI
- Interactive Navigation Bar
- Hero Section
- Food Menu with Add to Cart functionality
- Shopping Cart
- Customer Reviews Slider
- Food Delivery Services Section
- Mobile App Promotion Section
- Newsletter Subscription
- Attractive Footer
- Mobile-Friendly Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Swiper.js
